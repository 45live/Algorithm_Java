# 动态规划

## 要点

1. 从已知子问题的解，推导出当前问题的解。推导过程可以表达为一个数学公式。
2. 用一维、二维或多维数组来保存之前的计算结果。

## Dynamic Programming - 由Bellman提出

- **动态**：指用数学方法根据子问题求解当前问题，也即找到递推公式。
- **编程**：表示缓存上一步的结果，根据上一步的结果计算当前结果。

**注意：**

- 可能存在多种递推公式。创建数组时要考虑数组的初始化。
- 创建数组时，要根据题意明确数组含义和下标索引。

### 总结

- 找出递推公式，将当前问题分解成子问题，分阶段求解。
- 缓存子问题的解，在求解过程中避免重复计算。
